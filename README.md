# fhem-ftui-user-demos
<p>User demos for Fhem FTUI <a href="http://fhem.de/fhem.html">    http://fhem.de/fhem.html</a></p>

<p>Ihr findet hier im folgenden Fhem-FTUI-User-Demos. Diese sollen euch 

<p><strong>1. Sinn des Threads:</strong></p>

<li>Zeigen was mit FTUI möglich ist</li>
<li>Sein eigenes UI präsentieren und anderen Usern Anregungen und Hilfe bieten.</li>
<li>Niemand soll das Rad zum zweiten Mal erfinden müssen</li>
<li>Neuen Usern den Einstieg mit zahlreichen Beispielen erleichtern</li>


<p><strong>2. Wie kann ich anderen meine FTUI zur Verfügung stellen?</strong></p>

<li>Screenshots von den FTUI Seiten erstellen ggf. als zip/rar zusammenfassen.</li>
<li>html Seiten/Code am besten als zip/rar zusammenfassen.</li>
<li>BITTE auf Urheberrechtsverletzungen achten und ggf. vorher entfernen. Besonders bei Bildern/Logos über deren Rechte man nicht verfügt.</li>
<li>Zip/Rar Files und Screenshots hier im Thread posten oder per Email an mich.</li>
<li>Ich werde den Code und die Screenshots zusammen mit setstate sichten und allen hier und in einem gesonderten Git zur Verfügung stellen.</li>
<li>Es ist natürlich an euch selbst, die Demos hin und wieder durch senden eines aktuellen Standes aktuell zu halten.</li>


<p>Der Code in diesem Repository ist immer als Momentaufnahme zu verstehen, da die meisten User ihren Code stetik weiter entwickeln. Ist ist an den Usern hier öfters mal Aktualisierungen zu senden. ;-)</p>
